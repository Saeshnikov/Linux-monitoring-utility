# Linux Monitoring Utility.

## Options
### -h <br />
flags list <br /> 

### -t <br />
One bpftrace script working time <br />
default: 3600

### -T <br />
Program working time <br />
default: 86400

### -c <br />
Path to .yaml config file <br />
default: ../configs/defaultConf.yaml <br />

### -o  <br />
Path to the result <br />
default: . <br />
